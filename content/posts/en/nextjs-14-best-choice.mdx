## Key Advantages

1. **SSR & SSG** - Server-side rendering and static generation
2. **App Router** - Advanced routing system
3. **Server Actions** - Server-side form handling
4. **Image Optimization** - Automatic image optimization
5. **API Routes** - Easy API creation

## When NOT to Use Next.js?

<Callout type="warning">
Next.js isn't always the right answer. If you're building a simple application without SEO needs, regular React might be enough.
</Callout>

## Summary

If you're looking to build a modern website or application, Next.js is the safe choice. It provides all the tools you need for success.
